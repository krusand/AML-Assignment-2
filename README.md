This is the repository for the second mini-project of the DTU course [02460 Advanced Machine Learning](https://kurser.dtu.dk/course/2025-2026/02460?menulanguage=en). 

The project is about geodesics (shortest paths) in latent spaces of VAEs. The data used for the project is a subset of non-binarized MNIST.
